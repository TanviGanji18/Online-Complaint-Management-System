# Online Complaint Management System

This project is a Streamlit-based web application that allows users to register
complaints and enables administrators to manage and track complaint status.

## Features
- User complaint registration
- Auto-generated complaint ID
- Admin dashboard
- Complaint status update and search

## Technologies Used
- Python
- Streamlit
- MySQL (MySQL Workbench â€“ localhost)

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run user application:
   streamlit run app.py
3. Run admin application:
   streamlit run admin.py
